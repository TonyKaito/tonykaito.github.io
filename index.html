About

I'm eager to learn and grow, continually seeking opportunities to expand my computer knowledge and skillset. To see what I have done, explore my Github portfolio to see examples of my projects and feel free to connect with me through email.

Experience and Projects

Here are some of the projects I've worked on. A more detailed list (not exhaustive due to contracts) can be found on my Github portfolio.

West Nile Virus Susceptibility Analysis

West Nile Virus is a mosquito-borne disease that emerged in Europe from Africa in the early 1960s, and has spread to multiple countries. Created and trained a Tensorflow/Keras deep learning model to predict, given location of cases within Chicago, the most susceptible regions the virus is likely to spread.

SSIS Packages for OLTP Data Transfer and Integration

Created two SSIS packages that perform OLTP tasks, one exporting transaction data from a database into multiple flat files of a certain size with metadata contained within the file, and the other importing the said flat file into the database. This is to be used to transfer transaction data between multiple server locations.

Employee Satisfaction Analysis

Used a dataset/survey of 15000 employees' satisfactory experiences, performance reviews, and other work-related data and performed a descriptive analysis of the data as well as diagnostic, exploratory, and inferential analyses of the major factors affecting self-reported satisfaction levels.

ETL Integration of EDI Transactions into SQL Server Database

Created an ETL package in SSIS that transforms EDI transactions (XML data encoded into ASC X-12 format) into a SQL Server database. Modified an existing SQL Server stored procedure to perform the transformation, reducing runtime from an average of 45 minutes to less than a few seconds.

Japanese OCR Auto-translator

A small application that uses OpenCV to capture hard-subtitled Japanese text and Tesseract OCR to detect and grab the Japanese text within the screenshot. Then use an LLM model alongside Ollama or a translating transformer given by Hugging Face to auto-correct errors within the OCR and translate the Japanese text into English.

PDF-to-PNG Discord Bot

Created a Discord bot that converts a user-submitted PDF into PNGs so that Discord can embed the pages without having to download the file. Created Using the official Discord.py API alongside the Fitz library, and hosted on AWS using Amazon EC2.
